# Urban Report - Plateforme de signalement des problèmes urbains

## 📌 Description du projet
Urban Report est une plateforme web qui permet aux citoyens de signaler les problèmes urbains dans leurs villes afin d’améliorer la communication avec les autorités locales et accélérer les interventions.

Les utilisateurs peuvent signaler :
- nids-de-poule
- déchets dans les rues
- éclairage public en panne
- dégradation des infrastructures

Chaque signalement contient une description, une photo et une localisation géographique.

---

## 🎯 Objectifs du projet
- Faciliter le signalement des problèmes urbains
- Améliorer la communication citoyen / administration
- Suivre l’état des interventions
- Utiliser la géolocalisation pour plus de précision
- Permettre une gestion efficace des signalements

---

## 🧱 Architecture du projet

Le projet est composé de 3 parties :

- **Backend** : Spring Boot (API REST)
- **Frontend** : React JS
- **Base de données** : MySQL

---

## ⚙️ Fonctionnalités

### 👤 Utilisateur
- Inscription / Connexion
- Créer un signalement
- Ajouter photo
- Localisation du problème
- Suivre l’état du signalement
- Ajouter un avis

### 🛠 Administration
- Gestion des utilisateurs
- Gestion des signalements
- Changement de statut
- Validation des contenus

---

## 🔐 Sécurité

  - ADMIN_VILLE
  - SUPER_ADMIN
